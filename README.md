## Project setup

- Install dependencies with npm

  ```bash
    npm install
  ```
- Run docker
  ```bash
    npm run docker
  ```
- Create `.env` file and refer to `.env-sample` file
  ```env
  // Sample values
  DB_USERNAME=root
  DB_PASSWORD=root_password
  DB_DATABASE=database_name
  DB_HOST=127.0.0.1
  DB_PORT=3306
  APP_PORT=3001
  ```
- Run the app 
  ```bash
  npm run dev
  ```
